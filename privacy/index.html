1. 개요

Run Pet(「런 펫」, 이하 「앱」)은 실외·실내 러닝 기록, 펫 성장, 컬렉션, 챌린지 등 게임형 러닝 기능을 제공하는 모바일 애플리케이션입니다.

본 방침은 앱이 어떤 정보를 어떻게 이용·보관하는지 설명합니다. MVP(1.0.0) 기준 별도의 Run Pet 계정 로그인이나 Run Pet 운영 서버 동기화는 제공하지 않습니다. 주요 데이터는 사용자 기기 내에 저장됩니다.

1.1 의료·건강 관련 안내

Run Pet은 운동 기록 및 게임형 동기 부여를 위한 앱이며, 의료 진단, 치료, 질병 예방 목적의 서비스가 아닙니다. 앱에 표시되는 거리·페이스·칼로리 추정치·걸음 수 등은 참고용이며, 건강 상태 판단이나 의료 행위를 대체하지 않습니다.

3. 정보 이용 목적

  • 러닝 세션 기록·저장·표시
  • 펫 성장, 일일 퀘스트, 챌린지, 컬렉션 등 게임형 기능 제공
  • 사용자 설정(언어, 테마, 오디오 안내 등) 유지
  • Android 실외 러닝 백그라운드 기록을 위한 포그라운드 서비스 알림 표시(해당 OS·권한 허용 시)

광고 타깃팅, 행동 기반 광고, Run Pet 서버 분석 목적의 개인 식별 수집은 MVP에서 하지 않습니다.

4. 제3자 제공 및 외부 서비스

4.1 지도 서비스 (Google Maps / Mapbox)

앱은 경로·지도 표시를 위해 지도 제공 API를 사용할 수 있습니다.

  • Google Maps Platform (기본 구성): 지도 타일, 정적/동적 지도, 경로 관련 API 호출 시 위치·경로 관련 데이터가 Google에 전달될 수 있습니다.
  • Mapbox (운영자가 토큰을 설정한 경우): 동일하게 지도 API 제공자에게 요청 데이터가 전달될 수 있습니다.

Run Pet은 MVP 기준 지도 API 호출을 위해 필요한 범위에서만 해당 제공자와 데이터를 주고받습니다. Google/Mapbox 각사의 개인정보 처리방침이 별도로 적용됩니다.

  • Google Maps Platform: https://policies.google.com/privacy
  • Mapbox: https://www.mapbox.com/legal/privacy

4.2 앱 스토어 플랫폼

앱 배포·(향후)결제·OS 백업·플랫폼 기본 크래시 리포트 등은 Apple App Store / Google Play 정책 및 해당 플랫폼이 수집하는 정보에 따릅니다.

4.3 Run Pet 서버

MVP 1.0.0에는 Run Pet이 운영하는 중앙 서버로 사용자 데이터를 동기화하지 않습니다.

마이크: Run Pet MVP는 러닝 오디오 안내에 OS TTS(텍스트 음성 변환)를 사용하며, 마이크 녹음 권한을 요청하지 않습니다.

7. 보관 기간 및 삭제

  • 데이터는 사용자 기기 내에 보관됩니다.
  • 앱 삭제(제거) 시, 일반적으로 해당 앱 샌드박스의 로컬 데이터·캐시·첨부 사진이 함께 삭제됩니다.
  • OS 또는 기기 제조사 백업(iCloud, Google 백업 등)에 포함될 수 있으며, 이는 플랫폼 백업 정책에 따릅니다.
  • 개발·QA용 GPS trace 파일은 디버그 빌드에서만 생성·열람 UI가 노출되며, 릴리스 빌드 UI에서는 GPS trace 디버그 도구를 제공하지 않습니다.

10. 보안

Run Pet은 기기 내 저장 데이터에 대해 OS가 제공하는 앱 샌드박스·암호화 기능에 의존합니다. MVP는 별도 End-to-End 암호화 서버 동기화를 제공하지 않습니다.

11. 국외 이전

MVP 기준 Run Pet 자체 서버 이전은 없습니다. 지도 API(Google/Mapbox) 이용 시 해당 제공자의 서버(해외 포함)로 요청이 전달될 수 있습니다.

12. 정책 변경

본 방침은 법령·서비스 변경에 따라 수정될 수 있습니다. 중요한 변경 시 앱 또는 공개 URL을 통해 안내합니다. 최종 수정일은 문서 상단에 표시합니다.
