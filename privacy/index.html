제1조 (개인정보의 처리 목적)

Run Pet(런 펫)은 실외·실내 러닝 기록과 펫 성장 게임 기능을 제공하는 모바일 앱입니다. V1 출시는 주요 데이터를 사용자 기기에 저장하며 Run Pet 서버로 동기화하지 않습니다.

제2조 (처리하는 개인정보 항목)

  • 정밀 GPS / 경로 좌표 — 실외 러닝 시작 시 경로·거리·페이스·시간 기록. 활성 실외 러닝 중 OS 권한·설정이 허용하면 백그라운드에서도 처리될 수 있습니다.
  • 러닝 지표 및 만보기 데이터 — 기록, 실내 러닝, 일일 목표 및 게임 진행.
  • 사용자 선택 사진 — 기록에 사진을 첨부할 때만.
  • 펫·알·챌린지·인벤토리·배지·설정 데이터 — 로컬 게임 및 환경설정.
  • 게스트 프로필 설정·구독 자격 플래그 — SharedPreferences(IAP 미연동).

기기 내 처리와 기기 외 전송은 다릅니다. V1은 기록을 Run Pet 서버로 보내지 않지만, 지도 타일 표시 시 구성된 지도 제공자에 네트워크 요청이 발생할 수 있습니다.

제3조 (개인정보의 처리 및 보유 기간)

기록·게임 데이터는 기기 내 Drift/SQLite(schema v21), 설정은 SharedPreferences, 선택 사진은 앱 로컬 문서 저장소에 보관됩니다.

원시 GPS 경로는 최대 365일 보관 후 삭제(비위치 러닝 요약은 유지될 수 있음)합니다. 실외 러닝 전에는 현행 위치기반서비스 이용약관에 별도 동의합니다.

제4조 (개인정보의 제3자 제공)

V1에는 Run Pet 원격 계정·서버 동기화가 없습니다. 운영자가 Google Maps API 키를 설정한 경우 지도 타일 요청에 Google 개인정보 처리방침이 적용될 수 있습니다.

제5조 (개인정보 처리업무의 위탁 및 SDK 현황)

  • 친구 관계 — 기능 셸만 존재, 서버 미연동.
  • 광고 / IDFA — 현재 출시 빌드에서는 광고 게재가 활성화되지 않습니다. 앱 바이너리에 광고 SDK가 포함될 수 있으나, V1 출시 설정상 광고 요청·개인 맞춤 광고는 사용하지 않습니다. ATT 미사용.
  • Firebase Analytics / Remote Config — 프로젝트 미구성 시 초기화하지 않음.
  • 결제(StoreKit / Play Billing) — 판매 미연동.
  • 음성 안내 — 기기 내 TTS/네이티브 엔진 사용. 마이크 녹음·음성 데이터 서버 전송 없음.

제6조 (개인정보의 파기 및 이용자 통제)

러닝 중지·취소, OS 위치·사진·알림 권한 철회, 기록 삭제, 위치약관 동의 철회(저장 경로 전체 삭제), 앱 삭제가 가능합니다. V1 서버 계정이 없으므로 서버 계정 삭제는 해당 없음(NOT APPLICABLE).

제7조 (앱 접근 권한)

  • 위치(사용 중 / 백그라운드) — 실외 러닝 GPS
  • 모션 / 활동 인식 — 실내 만보기
  • 사진 라이브러리 — 기록 사진 첨부
  • 알림 — Android 실외 포그라운드 서비스(해당 시)

제8조 (미성년자)

Run Pet은 만 14세 이상을 대상으로 합니다. 앱은 로컬 자가확인 시각만 저장하며 생년월일·법정대리인 정보를 수집하지 않습니다. 해외 제공 여부와 관계없이 이용자 거주지의 강행 보호 규정이 적용될 수 있습니다.
