現状

アプリ内アカウント削除

設定 → アカウント → アカウント削除で確認後、ログインアカウント基準の端末内データを削除できます。元に戻せません。

サーバーアカウント削除

PENDING — release auth およびサーバー保存が有効化されるまでリモート削除 API は未接続です。

その他のローカル削除

  • アプリデータ全体 — アプリ削除
  • ラン記録・写真 — 記録単位削除（実装範囲）
  • ログアウト — 認証セッションのみ整理（ローカルデータ保持）
