概要

Run Pet（ランぺット）は屋外・屋内ラン記録とペット育成ゲームを提供するモバイルアプリです。14歳以上の方のみご利用いただけます。Apple または Google ログインでご利用いただけます。Run Pet 運営サーバーへユーザーデータを同期しません。主なデータは端末内に保存されます。

収集・保存する情報

  • アカウント識別子 — ログイン時。端末内データの範囲に使用。Run Pet サーバー送信なし。
  • メール・名前 — ログインプロバイダが返す場合あり。アプリに別途保存しない場合があります。
  • GPS / ルート座標 — 屋外ラン。端末内アプリ保存領域。
  • 走行距離・時間・ペース — 端末内のみ。
  • 歩数計 / 歩数 — 屋内ランと記録タブ。端末内アプリ保存領域。
  • ペット・卵・チャレンジ状態 — 端末内アプリ保存領域と設定。
  • ラン記録写真 — ユーザー選択（Plus）。
  • 必須同意・ログイン進入状態 — SharedPreferences（14歳以上、規約、プライバシー、位置情報サービス規約）。
  • サブスク資格フラグ — ローカル SharedPreferences（IAP 未連携）。

送信しない項目（現行リリース）

  • フレンド関係 — 機能シェルのみ。バックエンドオフ。
  • 広告 / IDFA — 現行リリースでは広告配信無効。SDK はバイナリに含まれる場合あり。ATT 未使用。
  • Firebase Analytics — 有効化されていません。
  • Firebase Remote Config — プロジェクト未構成時は初期化しません。
  • 決済データ — StoreKit / Play Billing 未連携。

音声案内

端末内 TTS/ネイティブエンジン。マイク録音なし。サーバー送信なし。

地図と第三者

ルート座標は端末に保存。Google Maps API キー設定時は Google へのネットワーク要求の可能性あり。

権限

  • 位置情報 — 屋外ラン GPS。屋外ラン開始時に OS 位置権限を要求。位置情報サービス規約は初回必須同意に含む。
  • モーション / 活動認識 — 屋内歩数計
  • フォトライブラリ — 記録写真
  • 通知 — Android 屋外 FGS（該当時）

データ削除

設定 → アカウント → アカウント削除でログインアカウント基準の端末内データを削除できます。ログアウトはセッションのみ整理しローカルデータは保持します。アプリ削除で全データ削除。サーバーアカウント削除は現時点アプリでは別途提供しておらず、導入時は本ページで案内します。
