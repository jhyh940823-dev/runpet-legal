第1条（個人情報の処理目的）

Run Pet（ランぺット）は屋外・屋内ラン記録とペット育成ゲーム機能を提供するモバイルアプリです。V1 リリースでは主なデータをユーザーの端末に保存し、Run Pet サーバーへ同期しません。

第2条（処理する個人情報の項目）

  • 精密 GPS / ルート座標 — 屋外ラン開始時、経路・距離・ペース・時間の記録。アクティブな屋外ラン中は OS 権限・設定が許可する場合バックグラウンドでも処理されることがあります。
  • ランニング指標・歩数計データ — 記録、屋内ラン、日次目標、ゲーム進行。
  • ユーザー選択写真 — 記録に写真を添付した場合のみ。
  • ペット・卵・チャレンジ・インベントリ・バッジ・設定 — ローカルゲーム・設定。
  • ゲストプロフィール設定・サブスク資格フラグ — SharedPreferences（IAP 未連携）。

端末内処理と端末外送信は異なります。V1 は記録を Run Pet サーバーへ送りませんが、地図タイル表示で構成された地図提供者へのネットワーク要求が発生する場合があります。

第3条（個人情報の処理および保有期間）

記録・ゲームデータは端末内 Drift/SQLite（schema v21）、設定は SharedPreferences、選択写真はアプリのローカルドキュメント保存に保管されます。

生の GPS 経路は最大 365 日保存後に削除（非位置ラン要約は残る場合があります）。屋外ラン前に現行の位置情報サービス利用規約への別途同意が必要です。

第4条（個人情報の第三者提供）

V1 には Run Pet リモートアカウント・サーバー同期がありません。運営者が Google Maps API キーを設定した場合、地図タイル要求に Google のプライバシーポリシーが適用される場合があります。

第5条（処理業務の委託および SDK 状況）

  • フレンド関係 — 機能シェルのみ。バックエンドオフ。
  • 広告 / IDFA — 現行リリースビルドでは広告配信は有効化されていません。アプリバイナリに広告 SDK が含まれる場合がありますが、V1 リリース設定では広告リクエスト・パーソナライズ広告は使用しません。ATT 未使用。
  • Firebase Analytics / Remote Config — プロジェクト未構成時は初期化しません。
  • 決済（StoreKit / Play Billing） — 販売未連携。
  • 音声案内 — 端末内 TTS/ネイティブエンジン。マイク録音・音声データのサーバー送信なし。

第6条（個人情報の破棄および利用者の管理）

ラン停止・キャンセル、OS 位置・写真・通知権限の撤回、記録削除、位置規約同意の撤回（保存経路の全削除）、アプリ削除が可能です。V1 サーバーアカウントがないため、サーバーアカウント削除はNOT APPLICABLEです。

第7条（アプリのアクセス権限）

  • 位置情報（使用中 / バックグラウンド）— 屋外ラン GPS
  • モーション / 活動認識 — 屋内歩数計
  • フォトライブラリ — 記録への写真添付
  • 通知 — Android 屋外フォアグラウンドサービス（該当時）

第8条（未成年者）

Run Pet は 14 歳以上を対象とします。アプリはローカル自己確認時刻のみ保存し、生年月日・法定代理人情報は収集しません。海外提供の有無にかかわらず、利用者の居住地の強行規制が適用される場合があります。

第9条（個人情報保護責任者および問い合わせ）

カスタマーサポートメール: support.runpet@gmail.com

第10条（処理方針の変更）

処理方式、サーバー同期、広告、分析、購入、アカウント、HealthKit/Health Connect、経路共有、児童政策などの重要な変更は、新たなレビューと通知後に有効化します。
