概要

Run Pet（ランぺット）は屋外・屋内ラン記録とペット育成ゲームを提供するモバイルアプリです。現行リリースビルドではRun Pet 運営サーバーへユーザーデータを同期しません。主なデータは端末内に保存されます。

収集・保存する情報

  • GPS / ルート座標 — 屋外ラン。端末内 Drift DB。Run Pet サーバー送信なし。
  • 走行距離・時間・ペース — 端末内のみ。
  • 歩数計 / 歩数 — 屋内ランと記録タブ。Drift / SharedPreferences。
  • ペット・卵・チャレンジ状態 — 端末内 Drift と設定。
  • ラン記録写真 — ユーザー選択（Plus）。アプリドキュメントフォルダ。
  • ゲストプロフィール設定 — SharedPreferences。
  • サブスク資格フラグ — ローカル SharedPreferences（IAP 未連携）。

送信しない項目（現行リリース）

  • フレンド関係 — 機能シェルのみ。バックエンドオフ。
  • 広告 / IDFA — 現行リリースビルドでは広告配信は有効化されていません。アプリバイナリに広告 SDK が含まれる場合がありますが、V1 リリース設定では広告リクエスト・パーソナライズ広告は使用しません。ATT 未使用。
  • Firebase Analytics — 有効化されていません。
  • Firebase Remote Config — プロジェクト未構成時は初期化しません。
  • 決済データ — StoreKit / Play Billing 未連携。

音声案内

ラン中の距離・時間などの案内に、端末内の音声合成（TTS/ネイティブエンジン）を使用する場合があります。マイク録音は行わず、音声データを Run Pet サーバーへ送信しません。

地図と第三者

ルート座標は端末に保存されます。運営者が Google Maps API キーを設定した場合、地図タイル表示で Google サービスへのネットワーク要求が発生する場合があります。

権限

  • 位置情報（使用中 / バックグラウンド）— 屋外ラン GPS
  • モーション / 活動認識 — 屋内歩数計
  • フォトライブラリ — 記録への写真添付
  • 通知 — Android 屋外フォアグラウンドサービス（該当時）

データ削除

アプリ削除またはアプリ内記録削除で端末データを削除できます。サーバーアカウント削除は現時点NOT APPLICABLEです。
