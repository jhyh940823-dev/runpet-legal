位置情報の利用

Run Pet は、利用者が明示的に開始した屋外ランニング中にGPSを使用し、経路・距離・時間・ペースを計算します。アクティブなランニング中は、画面ロックや他アプリ利用中でもOSが許可する範囲で位置情報を使用することがあります。

保管・削除・撤回

生のGPS経路は端末内に最大1年間保存されます。V1ではRun Petサーバーへの送信および第三者提供は行いません。利用者は個別記録を削除するか、設定で位置情報利用への同意を撤回できます。撤回時は保存済みの個人位置情報経路を消去し、新しい屋外GPSランニングは開始できません。ゲーム・ペット・クエスト・要約データは維持されます。

利用者への案内

Run Pet は14歳以上の方のみご利用いただけます。GPSには誤差があり、医療・緊急救助・ナビゲーションサービスではありません。
