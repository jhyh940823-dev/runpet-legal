第1条（目的）

本規約は、Run Pet が提供する位置情報サービス（屋外ラン GPS 記録）の利用条件を定めます。

第2条（規約の効力）

本規約はアプリ内の公開 URL および WebView を通じて提供されます。文書上部の最終更新日が適用されます。

第3条（サービスの内容）

Run Pet は、利用者が明示的に開始した屋外ランニング中に GPS を使用し、経路・距離・時間・ペースを計算します。アクティブなランニング中は、画面ロックや他アプリ利用中でも OS が許可する範囲で位置情報を使用することがあります。屋内ランニングは位置情報なしで利用できます。

第4条（個人位置情報の保有および利用期間）

生の GPS 経路は端末内に最大 1 年間保存されます。V1 では Run Pet サーバーへの送信および第三者提供は行いません。

第5条（個人位置情報主体の権利）

利用者は個別記録を削除するか、設定で位置情報利用への同意を撤回できます。撤回時は保存済みの個人位置情報経路を消去し、新しい屋外 GPS ランニングは開始できません。ゲーム・ペット・クエスト・要約データは維持されます。

第6条（位置情報管理責任者）

カスタマーサポートメール: support.runpet@gmail.com

第7条（免責および利用上の限界）

Run Pet は 14 歳以上の方のみご利用いただけます。GPS には誤差があり、医療・緊急救助・ナビゲーションサービスではありません。

第8条（お問い合わせ）

カスタマーサポートメール: support.runpet@gmail.com
