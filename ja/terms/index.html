第1条（目的）

Run Pet はユーザーの端末でランニング記録とペット動機付け機能を提供するモバイルアプリです。V1 は無料で利用でき、リアルタイム IAP/サブスク販売やサーバーアカウントを提供しません。将来の有料機能やアカウントは、別途有効化・開示されるまで本バージョンに含まれません。

第2条（サービスの提供）

GPS、歩数計、ペース、距離、カロリー、ルート、音声案内は便宜機能です。医療・緊急救助・ナビゲーション・フィットネス安全サービスではありません。利用者は安全な屋外活動の判断を自ら行い、端末・権限設定を適切に保ち、危険な状況では中止する必要があります。

屋外 GPS ランニングは、OS 位置権限の要求前に現行の位置情報サービス利用規約への別途同意が必要です。

第3条（利用資格およびアカウント）

Run Pet は 14 歳以上の利用者を対象とします。法定代理人同意の確認手続きを提供せず、そのような確認を提供していると表示しません。

V1 には Run Pet サーバーアカウントがありません。

第4条（利用者の義務）

利用者はサービスを誤用したり運用を妨害したりしてはならず、法令で禁止される場合のリバースエンジニアリングや他人の権利侵害をしてはなりません。

第5条（コンテンツおよびデータ）

現行リリースは主なデータを端末内に保存し、Run Pet サーバー同期を提供しません。

距離・ペース・歩数は GPS と端末センサーによる推定値です。位置・センサー結果は不正確・遅延・未提供・損失となる場合があります。

生の GPS 経路は端末内に最大 365 日保存され、記録削除または位置規約同意撤回でより早く削除できます。削除・端末紛失・OS 動作・再インストール・バックアップ不可などでローカルデータが失われる場合があり、V1 は Run Pet クラウドバックアップを提供しません。

第6条（有料サービスおよび広告）

Run Pet Plus サブスクの特典・フード政策はサブスクページを参照。自動更新・返金は Billing 連携後にストア規約に従い確定 （ストア登録前の価格・商品情報）。

第7条（Run Pet の権利）

Run Pet は適用法令の範囲内でアプリコンテンツに関する権利を保有します。

第8条（責任の制限）

故意・重過失など法令上排除できない責任は除外しません。責任制限・準拠法・管轄・消費者紛争に関する文言は、最終的な運営者・法務レビューが必要です。

第9条（準拠ルールおよび紛争解決）

本規約およびサービス利用には、大韓民国の関連法令およびアプリマーケットのポリシーが適用されます。紛争が発生した場合は、まずサポート窓口を通じて解決を試み、必要に応じて関連法令で定められた手続きに従います。
