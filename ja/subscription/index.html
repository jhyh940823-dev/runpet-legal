Run Pet Plus

Run Pet Plus は月額/年額サブスクリプション提供予定。プロダクト ID・価格は ストア登録前の価格・商品情報。

特典（正確に4つ）

  1. すべての広告を非表示
  2. ランニング記録に写真を追加
  3. 毎日追加フード2個を自動付与
  4. 週間クエスト達成時にR通貨+1

フード政策

1日のフード上限は3個。残りは翌日に消えます。サブスクの便利特典は日次上限を引き上げません。

現行ビルド

  • StoreKit / Play Billing — 未連携
  • 購入 CTA — 準備中（fail-closed）
  • AdMob — リリースビルドで広告配信無効（SDK 同梱の可能性あり）。広告非表示特典は政策上予定

サブスク管理

IAP 連携後、iOS/Android のサブスク設定での解約・復元案内を提供します。

未成年者の決済

未成年者が法定代理人の同意なく有料サービスを決済した場合、本人または法定代理人は関連法令に基づき決済を取消できる場合があります。
