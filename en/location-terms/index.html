Location use

During an outdoor run explicitly started by the user, Run Pet uses GPS to calculate route, distance, time, and pace. While an active run continues, location may be used within OS-permitted limits when the screen is locked or another app is open.

Retention, deletion, and withdrawal

Raw GPS route geometry is retained on the device for up to one year. V1 does not send routes to a Run Pet server or provide them to third parties. Users can delete individual records or withdraw location consent in Settings. Withdrawal erases stored personal-location route geometry and prevents new outdoor GPS runs; game, pet, quest, and summary data remain.

User notice

Run Pet is available only to users aged 14 and over. GPS can be inaccurate and is not a medical, emergency, rescue, or navigation service.
