Overview

Run Pet is a mobile app combining outdoor/indoor run logging with pet growth gameplay. Users must be 14 years of age or older. The launch target requires Apple or Google sign-in; guest play is not offered. Release builds may keep auth disabled until operator activation. User data is not synced to a Run Pet operator server in the current release. Primary data stays on your device.

Data collected and stored

  • Account identifier (OAuth provider uid) — when signed in; scopes on-device data; no Run Pet server upload in current release.
  • Email / name — OAuth providers may return these; Run Pet may not store them separately in app DB. Operator confirm for store disclosures.
  • GPS / route points — outdoor runs; on-device Drift DB; no Run Pet server upload.
  • Run distance, time, pace — on-device only.
  • Pedometer / steps — indoor runs and Records tab.
  • Pet, egg, challenge state — on-device Drift and preferences.
  • Run record photos — user-selected (Plus).
  • Required consent and login-entry state — SharedPreferences (14+, terms, privacy, location-based service terms).
  • Subscription entitlement flag — local SharedPreferences (IAP not connected).

Not transmitted (current release)

  • Friend relations — feature shell only; backend off.
  • Ads / IDFA — Ad serving disabled in current release builds. SDK may be in binary; no personalized ads. No ATT.
  • Firebase Analytics — not active.
  • Firebase Remote Config — not initialized when project not configured.
  • Payment data — StoreKit / Play Billing not connected.

Voice guidance

On-device TTS/native engine; no microphone recording; no server upload.

Maps and third parties

Route points stored locally. Map tiles may request Google when API key is configured.

Permissions

  • Location — outdoor run GPS. OS location permission is requested when starting an outdoor run. Location-based service terms are accepted in first-login required consent.
  • Motion / activity — indoor pedometer
  • Photo library — record photos
  • Notifications — Android outdoor foreground service when applicable

Data deletion

Settings → Account → Delete account removes on-device data for the signed-in account. Sign out keeps local run/pet data. Uninstall removes all local data. Remote server account deletion is PENDING until release auth ships.
