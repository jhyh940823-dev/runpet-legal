Overview

Run Pet is a mobile app combining outdoor/indoor run logging with pet growth gameplay. In current release builds, user data is not synced to a Run Pet operator server. Primary data stays on your device.

Data collected and stored

  • GPS / route points — outdoor runs; stored in on-device Drift DB; no Run Pet server upload.
  • Run distance, time, pace — on-device only.
  • Pedometer / steps — indoor runs and Records tab; Drift / SharedPreferences.
  • Pet, egg, challenge state — on-device Drift and preferences.
  • Run record photos — user-selected (Plus); app documents folder.
  • Guest profile preferences — SharedPreferences.
  • Subscription entitlement flag — local SharedPreferences (IAP not connected).

Not transmitted (current release)

  • Friend relations — feature shell only; backend off.
  • Ads / IDFA — AdMob not connected; no ATT.
  • Analytics (Firebase) — not active.
  • Payment data — StoreKit / Play Billing not connected.

Maps and third parties

Route points are stored locally. When an operator configures a Google Maps API key, map tiles may trigger network requests to Google services.

Permissions

  • Location (when-in-use / background) — outdoor run GPS
  • Motion / activity recognition — indoor pedometer
  • Photo library — attach photos to records
  • Notifications — Android outdoor foreground service when applicable

Data deletion

Remove on-device data by deleting the app or individual records in-app. Server account deletion is NOT APPLICABLE today.
