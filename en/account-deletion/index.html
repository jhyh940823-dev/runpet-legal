Account Deletion

Last updated: 2026-06-21

Privacy Policy Terms of Use Support Account Deletion Subscription

Current status

The launch target requires Apple or Google sign-in; guest play is not offered. Release builds may keep auth disabled until operator activation. Run Pet server account sync is not offered in the current release.

In-app account deletion

Settings → Account → Delete account shows a confirmation step, then deletes on-device run records, pets, challenges, and consent data for the signed-in account. This cannot be undone.

Server account deletion

PENDING — remote account deletion is not connected until release auth and server storage ship.

Other local deletion

  • All app data — uninstall the app
  • Run records / photos — per-record delete where implemented
  • Sign out — clears auth session; keeps local data

Relationship to this page

This page is public disclosure for stores and the web. Perform deletion in the app Delete account menu.
