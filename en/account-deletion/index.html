Account Deletion

Last updated: 2026-06-20

Privacy Policy Terms of Use Support Account Deletion Subscription

Current status

Default release mode is guest play. Run Pet server accounts and server-stored PII are not offered today.

Server account deletion

NOT APPLICABLE — no server account exists; no web/in-app server deletion path required.

When Sign-In and server storage ship, deletion UI/API/web URL will move from a separate process after server accounts are enabled to live.

Local data deletion (available now)

  • All app data — uninstall the app
  • Run records / photos — per-record delete in-app where implemented
  • Guest preferences — removed on uninstall
  • Auth session — logout clears local session (dev paths)

Store disclosure

Google Play and Apple require deletion paths when server accounts exist. We disclose guest-first, no server account honestly for the current release.
