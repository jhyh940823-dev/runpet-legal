위치정보 이용

Run Pet은 사용자가 명시적으로 시작한 실외 러닝 동안 GPS를 이용해 경로, 거리, 시간과 페이스를 계산합니다. 활성 러닝 중에는 화면 잠금 또는 다른 앱 사용 중에도 OS 허용 범위에서 위치를 사용할 수 있습니다.

보관·삭제·철회

원시 GPS 경로는 사용자 기기에 최대 1년 보관됩니다. Run Pet V1은 경로를 Run Pet 서버로 전송하거나 제3자에게 제공하지 않습니다. 사용자는 개별 기록을 삭제하거나 설정에서 위치정보 이용 동의를 철회할 수 있으며, 철회하면 저장된 개인위치정보 경로를 파기하고 새 실외 GPS 러닝을 시작할 수 없습니다. 게임·펫·퀘스트 요약 데이터는 유지됩니다.

이용자 안내

Run Pet은 만 14세 이상만 이용할 수 있습니다. GPS에는 오차가 있을 수 있으며 의료·긴급구조·내비게이션 서비스가 아닙니다.
