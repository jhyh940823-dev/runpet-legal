제1조 (목적)

본 약관은 Run Pet이 제공하는 위치기반서비스(실외 러닝 GPS 기록)의 이용 조건을 정합니다.

제2조 (약관의 효력)

본 약관은 앱 내 공개 URL 및 WebView를 통해 제공됩니다. 문서 상단의 최종 업데이트일을 기준으로 합니다.

제3조 (서비스의 내용)

Run Pet은 사용자가 명시적으로 시작한 실외 러닝 동안 GPS를 이용해 경로, 거리, 시간과 페이스를 계산합니다. 활성 러닝 중에는 화면 잠금 또는 다른 앱 사용 중에도 OS 허용 범위에서 위치를 사용할 수 있습니다. 실내 러닝은 위치 없이 이용할 수 있습니다.

제4조 (개인위치정보의 보유 및 이용기간)

원시 GPS 경로는 사용자 기기에 최대 1년 보관됩니다. Run Pet V1은 경로를 Run Pet 서버로 전송하거나 제3자에게 제공하지 않습니다.

제5조 (개인위치정보주체의 권리)

사용자는 개별 기록을 삭제하거나 설정에서 위치정보 이용 동의를 철회할 수 있습니다. 철회하면 저장된 개인위치정보 경로를 파기하고 새 실외 GPS 러닝을 시작할 수 없습니다. 게임·펫·퀘스트 요약 데이터는 유지됩니다.

제6조 (위치정보관리책임자)

  • 위치기반서비스 제공자: 멍스토어
  • 위치정보관리책임자: 박정현
  • 문의: support.runpet@gmail.com

제7조 (면책 및 이용 한계)

Run Pet은 만 14세 이상만 이용할 수 있습니다. GPS에는 오차가 있을 수 있으며 의료·긴급구조·내비게이션 서비스가 아닙니다.
