목적

본 약관은 Run Pet이 제공하는 위치기반서비스(실외 러닝 GPS 기록)의 이용 조건을 정합니다.

서비스 내용

Run Pet은 사용자가 명시적으로 시작한 실외 러닝 동안 GPS를 이용해 경로, 거리, 시간, 페이스를 계산합니다. 실내 러닝은 실외 위치약관 동의 없이 이용할 수 있습니다. 활성 실외 러닝 중에는 OS 허용 범위에서 백그라운드 위치를 사용할 수 있습니다.

개인위치정보 보유 및 이용

  • 원시 GPS routePoints는 사용자 기기에 최대 1년 보관됩니다.
  • Firebase 계정 백업(V1)에는 원시 GPS routePoints를 포함하지 않습니다.
  • iOS CloudKit 백업은 별도 정책·동의 범위가 적용될 수 있습니다.

이용자 권리 및 철회

  • 위치약관 동의 철회 시 저장된 routePoints를 삭제하며 러닝 요약은 유지될 수 있습니다.
  • 진행 중인 실외 러닝에서는 안전·기록 무결성을 위해 동의 철회가 제한될 수 있습니다(종료 후 철회 가능).
  • 개별 기록 삭제 및 OS 위치 권한 철회가 가능합니다.

제3자 제공

원시 GPS 경로를 제3자에게 판매·제공하지 않습니다. 지도 타일 표시 시 구성된 지도 제공자(Google Maps 등)에 네트워크 요청이 발생할 수 있으며, 해당 제공자의 정책이 적용될 수 있습니다.

위치기반서비스사업 신고

위치기반서비스사업 신고는 승인 완료되었습니다.
