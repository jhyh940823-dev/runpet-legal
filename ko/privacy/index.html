개요

Run Pet(런 펫)은 실외·실내 러닝 기록과 펫 성장 게임을 제공하는 모바일 앱입니다. 현재 출시 빌드 기준 Run Pet 운영 서버로 사용자 데이터를 동기화하지 않습니다. 주요 데이터는 사용자 기기에 저장됩니다.

수집·저장하는 정보

  • GPS / 경로 좌표 — 실외 러닝 시. 기기 내 Drift DB 저장. Run Pet 서버 전송 없음.
  • 러닝 거리·시간·페이스 — 기기 내 저장. 서버 전송 없음.
  • 만보기 / 걸음 수 — 실내 러닝 및 기록 탭. Drift / SharedPreferences.
  • 펫·알·챌린지 상태 — 기기 내 Drift 및 설정.
  • 러닝 기록 사진 — 사용자가 직접 선택(Plus). 앱 문서 폴더에 저장.
  • 게스트 프로필 설정 — SharedPreferences.
  • 구독 자격 플래그 — 로컬 SharedPreferences(IAP 미연동).

전송하지 않는 항목 (현재 출시)

  • 친구 관계 — 기능 셸만 존재, 서버 미연동.
  • 광고 / IDFA — 현재 출시 빌드에서는 광고 게재가 활성화되지 않습니다. 앱 바이너리에 광고 SDK가 포함될 수 있으나, V1 출시 설정상 광고 요청·개인 맞춤 광고는 사용하지 않습니다. ATT 미사용.
  • 분석(Firebase Analytics) — 활성화되지 않음.
  • Firebase Remote Config — 프로젝트 미구성 시 초기화하지 않음.
  • 결제 데이터 — StoreKit / Play Billing 미연동.

음성 안내

러닝 중 거리·시간 등 안내를 위해 기기 내 음성 합성(TTS/네이티브 엔진)을 사용할 수 있습니다. 마이크 녹음은 하지 않으며, 음성 데이터를 Run Pet 서버로 전송하지 않습니다.

지도 및 제3자

경로 좌표는 기기에 저장됩니다. 지도 타일 표시 시 운영자가 Google Maps API 키를 설정한 경우 Google 서비스에 네트워크 요청이 발생할 수 있습니다.

권한

  • 위치(사용 중 / 백그라운드) — 실외 러닝 GPS
  • 모션 / 활동 인식 — 실내 만보기
  • 사진 라이브러리 — 기록 사진 첨부
  • 알림 — Android 실외 포그라운드 서비스(해당 시)

데이터 삭제

앱 삭제 또는 앱 내 기록 삭제로 기기 데이터를 제거할 수 있습니다. 서버 계정 삭제는 현재 해당 없음(NOT APPLICABLE).
