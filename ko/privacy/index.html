개요

Run Pet(런 펫)은 실외·실내 러닝 기록과 펫 성장 게임 기능을 제공하는 모바일 앱입니다. 현재 출시 빌드(prod)는 Firebase Auth 로그인이 필수이며, 게스트 출시 로그인은 지원하지 않습니다. Apple 또는 Google 로그인을 사용합니다.

처리하는 개인정보 항목 및 목적

  • Firebase Auth 식별자(UID), Apple/Google 로그인 제공자 정보 — 계정 생성·인증, 기기-계정 연결(owner binding)
  • 이메일 또는 제공자가 전달한 연락 정보 — 계정 식별·복구(제공 시), 친구 기능 내부 식별(공개 UI에 노출하지 않음)
  • 공개 표시 이름 — 친구·참고용 활동 요약 등 공개 프로필 표시
  • 연령(만 14세 이상) 자가확인 결과 및 동의 버전·시각 — 법정 연령 제한 및 필수 약관 동의 증빙
  • 위치 동의 상태, 정밀 GPS / routePoints — 실외 러닝 경로·거리·페이스 기록(동의 시)
  • 러닝 요약·지표, 만보기 데이터 — 기록, 실내 러닝, 일일 목표, 게임 진행
  • 사용자 선택 사진 — 기록에 사진을 첨부할 때만(기기 로컬)
  • 펫·알·챌린지·설정·배지 상태 — 게임 및 환경설정
  • 친구 코드, 친구 관계, 친구 활동 요약 — 친구 추가·참고용 활동 요약(Firebase/Firestore)
  • Firebase 계정 백업(V1) 요약 상태 — 기기 변경 시 계정 연결 데이터 복원(원시 GPS·원본 사진 제외)
  • iOS CloudKit 개인 백업(해당 시) — iOS 사용자 선택 iCloud 백업/복원(별도 범위·동의 적용)
  • 크래시/진단 로그(해당 시) — 안정성 개선 — 출시 빌드에서는 민감정보 로그 차단 정책 적용

현재 출시에서 수집하지 않는 항목

  • 광고 식별자(AdMob 출시 통합 비활성)
  • IAP 영수증 원문(구매·복원 비활성)
  • 마케팅 푸시 수신 동의(기능 비활성)
  • 법정대리인 정보, 생년월일

저장 위치 및 외부 처리

  • 기기 로컬: Drift/SQLite, SharedPreferences, 앱 문서 저장소(사진 등)
  • Firebase / Google Cloud: Auth, Firestore(프로필·친구·백업 메타 등), Cloud Functions(계정 삭제 정리 등)
  • Apple: Sign in with Apple, CloudKit(iOS 백업 해당 시)
  • Google: Google Sign-In, Google Maps 타일(구성 시)
  • 광고(AdMob): 바이너리 포함 가능하나 현재 출시에서 광고 요청 비활성
  • IAP/구독: 현재 출시에서 구매·복원 비활성 — Apple/Google이 처리하는 거래 원문은 Run Pet 서버에 보관하지 않음

보유 기간 및 이용자 통제

  • 원시 GPS 경로: 최대 365일(기기), 위치약관 철회 시 routePoints 삭제·러닝 요약 유지 정책
  • 계정 탈퇴: Firebase Auth 계정 삭제, Firestore 프로필·친구 참조 정리, 로컬 Drift 데이터 삭제 경로 제공
  • 로그아웃: 로컬 Drift 데이터는 삭제하지 않음
  • 마케팅 푸시: 현재 비활성 — 향후 활성화 시 별도 명시적 동의 및 야간(21:00~08:00) 별도 동의 필요

미성년자

Run Pet은 만 14세 이상 전용입니다. 법정대리인 동의 절차를 제공하지 않으며, 만 14세 미만은 가입을 완료할 수 없습니다. OAuth 후 필수 동의 전에 생성된 임시 Auth 계정은 동의 거절·미성년 분기 시 삭제됩니다.

앱 접근 권한

  • 위치(사용 중 / 백그라운드) — 실외 러닝 GPS
  • 모션 / 활동 인식 — 실내 만보기
  • 사진 라이브러리 — 기록 사진 첨부
  • 알림 — Android 실외 포그라운드 서비스(해당 시)

음성 안내

러닝 중 거리·시간 등 안내를 위해 기기 내 음성 합성(TTS/네이티브 엔진)을 사용할 수 있습니다. 마이크 녹음은 하지 않으며, 음성 데이터를 서버로 전송하지 않습니다.

처리방침의 변경

처리 방식, 서버 동기화 범위, 광고, 구매, 계정, HealthKit/Health Connect, 위치 처리, 아동 정책 등 중요 변경은 새 검토 및 고지 후 활성화합니다.
