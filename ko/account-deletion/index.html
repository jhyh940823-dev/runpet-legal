서버 계정 삭제

NOT APPLICABLE — 서버 계정이 없으므로 웹/앱 내 서버 삭제 경로가 필요하지 않습니다.

향후 Sign-In 및 서버 저장이 활성화되면 서버 계정 연동 후 별도 절차 안내 예정 상태의 삭제 UI·API·웹 URL을 제공합니다.

로컬 데이터 삭제 (지금 가능)

  • 전체 앱 데이터 — 앱 삭제(기기에서 제거)
  • 러닝 기록·사진 — 앱 내 기록별 삭제(해당 기능 구현 범위)
  • 게스트 설정 — 앱 삭제 시 함께 제거
  • 인증 세션 — 로그아웃(로컬 세션 정리, dev 경로)

스토어 공시

Google Play / Apple은 서버 계정 제공 시 삭제 경로를 요구합니다. 현재는 게스트 우선·서버 미연동으로 정직하게 공시합니다.
