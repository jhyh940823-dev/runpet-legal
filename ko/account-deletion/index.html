현재 상태

Run Pet 출시 목표는 Apple 또는 Google 로그인이며 게스트 이용은 제공하지 않습니다. 현재 출시 빌드는 운영자 설정에 따라 인증이 비활성일 수 있습니다. Run Pet 운영 서버 계정 동기화는 현재 제공하지 않습니다.

앱 내 계정 삭제

설정 → 계정 → 계정 삭제에서 삭제 전 확인 절차 후, 로그인한 계정 기준 기기 내 러닝 기록·펫·챌린지·동의 정보를 삭제할 수 있습니다. 삭제 후에는 되돌릴 수 없습니다.

서버 계정 삭제

PENDING — Run Pet 서버에 계정이 저장·동기화되는 기능(release auth)이 활성화되기 전까지 원격 계정 삭제 API는 연결되지 않았습니다. 활성화 후 별도 절차를 안내할 예정입니다.

로컬 데이터 삭제 (기타)

  • 전체 앱 데이터 — 앱 삭제(기기에서 제거)
  • 러닝 기록·사진 — 앱 내 기록별 삭제(해당 기능 구현 범위)
  • 로그아웃 — 인증 세션 정리(로컬 데이터 유지)

공개 안내와의 관계

본 페이지는 스토어·웹 공시용 안내입니다. 실제 삭제는 앱 내 계정 삭제 메뉴에서 수행합니다.
