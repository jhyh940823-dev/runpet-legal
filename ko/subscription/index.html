Run Pet Plus

Run Pet Plus는 월간/연간 구독으로 제공될 예정입니다. 제품 ID·가격은 스토어 등록 전 가격·상품 정보 미확정입니다.

혜택 (정확히 4가지)

  1. 모든 광고 제거
  2. 러닝 기록에 사진 추가
  3. 매일 추가 먹이 2개 자동 지급
  4. 주간퀘스트 완료 시 R재화 1개 추가 지급

먹이 정책

하루 먹이는 최대 3개이며, 남은 먹이는 다음 날 사라집니다. 구독은 편의 혜택이며 일일 상한을 올리지 않습니다.

현재 빌드

  • StoreKit / Play Billing — 미연동
  • 구매 CTA — 준비 중(fail-closed)
  • AdMob — 출시 빌드에서 광고 게재 비활성(SDK 포함 가능). 광고 제거 혜택은 정책상 예정

구독 관리

실제 IAP 연동 후 iOS/Android 구독 설정에서 해지·복원 안내를 제공합니다.
