제1조 (목적)

Run Pet(런 펫)은 사용자 기기에 러닝 기록과 펫 동기부여 기능을 제공하는 모바일 앱입니다. V1은 무료로 이용할 수 있으며, 실시간 IAP·구독 판매 또는 Run Pet 서버 계정을 제공하지 않습니다. 향후 유료 기능이나 계정은 별도 활성화 및 고지 전까지 본 버전에 포함되지 않습니다.

제2조 (서비스의 제공)

GPS, 만보기, 페이스, 거리, 칼로리, 경로 및 음성 안내는 편의 기능입니다. 의료·긴급구조·내비게이션·피트니스 안전 서비스가 아닙니다. 이용자는 안전한 실외 활동 판단을 스스로 하고, 기기·권한 설정을 적절히 유지하며, 위험한 상황에서는 중단해야 합니다.

실외 GPS 러닝은 OS 위치 권한 요청 전에 현행 위치기반서비스 이용약관에 별도 동의해야 합니다.

제3조 (이용자격 및 계정)

Run Pet은 만 14세 이상 이용자를 대상으로 합니다. 법정대리인 동의 확인 절차를 제공하지 않으며, 그러한 확인을 제공한다고 표시하지 않습니다.

V1에는 Run Pet 서버 계정이 없습니다.

제4조 (이용자의 의무)

이용자는 서비스를 오용하거나 운영을 방해해서는 안 되며, 법령에서 금지하는 경우 역공학하거나 타인의 권리를 침해해서는 안 됩니다.

제5조 (콘텐츠 및 데이터)

현재 출시 빌드는 주요 데이터를 기기 내에 저장하며 Run Pet 서버 동기화를 제공하지 않습니다.

거리·페이스·걸음 수는 GPS 및 기기 센서 기반 추정치이며, 환경·기기·신호에 따라 부정확·지연·미제공·손실될 수 있습니다.

원시 GPS 경로는 최대 365일 보관되며, 기록 삭제 또는 위치약관 동의 철회로 더 일찍 삭제할 수 있습니다. 삭제·기기 분실·OS 동작·재설치·백업 불가 등으로 로컬 데이터가 손실될 수 있으며, V1은 Run Pet 클라우드 백업을 제공하지 않습니다.

제6조 (유료서비스 및 광고)

Run Pet Plus 구독 혜택·먹이 정책은 별도 구독 안내 페이지를 참고하세요. 자동 갱신·환불은 스토어 정책 및 Billing 연동 후 확정됩니다 (스토어 등록 전 가격·상품 정보 미확정).

제7조 (Run Pet의 권리)

Run Pet은 적용 법령 범위 내에서 앱 콘텐츠에 대한 권리를 보유합니다.

제8조 (책임 제한)

고의·중과실 등 법령상 배제할 수 없는 책임을 제외합니다. 책임 제한·준거법·관할·소비자 분쟁 관련 문구는 최종 운영자·법률 검토가 필요합니다.

제9조 (준거법 및 분쟁 해결)

본 약관과 서비스 이용에는 대한민국 관련 법령 및 앱 마켓 정책이 적용됩니다. 분쟁이 발생하면 고객지원 채널을 통해 먼저 해결을 시도하며, 필요한 경우 관련 법령에서 정한 절차에 따릅니다.

제10조 (문의)
